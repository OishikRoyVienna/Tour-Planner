# 📖 Overview
TourPlanner is a web-based application for planning and tracking tours (bike, hike, running, or vacation).
Users can create tours in advance and manage logs with statistical data of accomplished tours.

# Installation and Getting Started
## 1. Clone the repository
- git clone https://github.com/OishikRoyVienna/Tour-Planner.git
- cd your_path_where_you_cloned_the_repo

## 2. Install dependencies
- yarn install

## 3. Start the development server
- yarn start

## Access the Application
Open a browser of your choice and type in following URL: http://localhost:4200

# 👤 User Guide (How to use the Application)
## Step 1: Register
- Navigate to http://localhost:4200
- Click "Register" link
- Fill in credentials
- Click "Register" and you will be redirected to login

## Step 2: Login
- Login in with your credentials which you determined in the registration process

## Step 3: Create a Tour

- From Dashboard, click "+ New Tour" card
- Fill in tour details:
- Click "Create Tour"
- You will be redirected to Tour Detail view
